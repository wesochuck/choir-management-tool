<task type="auto">
  <name>Task 2: Implement Secure Quick RSVP Hook</name>
  <files>pocketbase/pb_hooks/quick_rsvp.pb.js</files>
  <action>
    Create a PocketBase JS hook that:
    1. Defines `POST /api/quick-rsvp`.
    2. Retrieves HMAC secret from `appSettings` (key: 'HMAC_SECRET'). If missing, use a safe default or log error.
    3. Verifies token payload vs signature (HMAC-SHA256).
    4. Payload structure: `eventId|profileId|choice|expiresAt`.
    5. On success, updates/creates `eventRosters` record for the profile/event using superuser DAO.
    6. Returns JSON: `{ success: true, choice, eventTitle }`.
  </action>
  <verify>
    <automated>ls pocketbase/pb_hooks/quick_rsvp.pb.js</automated>
  </verify>
  <done>Custom RSVP endpoint implemented.</done>
</task>
