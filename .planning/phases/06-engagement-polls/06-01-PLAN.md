    Create backend hooks to handle poll token generation and verification.
    Copy logic pattern from rsvp.pb.js.
    
    Implement:
    1. POST /api/generate-poll-tokens: Admins only. Accepts { pollId, profileIds }. Returns { tokens: { profileId: token } }. Token payload: l=pollId&p=profileId.
