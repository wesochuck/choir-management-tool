# Ignore Rehearsal RSVPs for Automated Reminders

For Rehearsals tied to a parent Performance, the system completely ignores the Rehearsal's own RSVP list when dispatching automated event reminders. Instead, reminders are sent exclusively to the "Yes (Attending)" roster of the parent Performance.

**Why:** A singer who is performing is generally expected to attend the associated rehearsals. By strictly tying the reminder audience to the parent Performance roster, we ensure that all active performers receive the logistical updates for rehearsals, trading per-rehearsal precision (such as respecting explicit rehearsal declines) for communication coverage.
